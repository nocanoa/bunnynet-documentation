---
title: IP Addresses
description: You can use the following IP addresses to whitelist Bunny.net traffic in your firewall or ACLs.
---

The API endpoint `https://api.bunny.net/mc/nodes` returns IP addresses as plain text (one per line) or JSON array depending on your `Accept` header.

<Info>
  The list returned from the API is dynamic, make sure to request it
  periodically to avoid missing newly added IPs.
</Info>

<RequestExample>

```bash text/plain
curl https://api.bunny.net/mc/nodes
```

```bash application/json
curl -H "Accept: application/json" https://api.bunny.net/mc/nodes
```

</RequestExample>

<ResponseExample>

```bash text/plain
104.166.147.46
109.61.83.105
109.61.83.248
```

```json application/json
["104.166.147.46", "109.61.83.105", "109.61.83.248"]
```

</ResponseExample>
