Magic Containers is an edge compute platform for deploying containerized applications globally. The platform automatically identifies user locations and deploys your applications close to them, with workloads scaling dynamically based on demand.

## Key features

- **Global deployment** - Deploy containers wherever and whenever you need them, with automatic placement near your users
- **AI-powered autoscaling** - Intelligent provisioning scales infrastructure based on real-time demand
- **Self-healing** - Failed nodes trigger automatic relocation to healthy nodes within the same region
- **Automatic restarts** - Out-of-memory containers are automatically restarted
- **Shared and reserved runtimes** - Choose between shared resources or reserved capacity for mission-critical workloads
