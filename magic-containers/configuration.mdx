# What you'll need

Before you dive in, make sure you have the following prerequisites in place:

- A [bunny.net](https://bunny.net/) account ([Log in](https://dash.bunny.net/auth/login?pk_buttonlocation=menu) or sign up for a [free trial](https://dash.bunny.net/auth/register)).
- Ensure that you have already set up the application you want to configure. See our Quickstart guide if you haven't set up an application yet.

# Editing app configuration

To edit the configuration of your app, follow the steps below: 

1. Login to [bunny.net dashboard](https://dash.bunny.net/auth/login?pk_buttonlocation=menu).

2. Navigate to the **Magic Containers** section and select the app you want to modify.

3. Click on **Container Settings** and click on **Edit**.

<Frame>
  ![](/images/docs/7e7018b9b95eae9ed2157305c047f802618299c40888dd8202562ecc5392be2c-image.png)
</Frame>

5. Within the edit window, you can modify various configuration parameters, such as environment variables, health checks, endpoints, etc.

<Frame>
  ![](/images/docs/b3926acc2c98c4d616bed5973d9cbdb3d442f0a7d9253457cfcdca0ff2eb36b3-image.png)
</Frame>

6. After making the necessary modifications, click on **Update Container** to apply the updated configuration.
