## Face crop

**_Parameter_**: face\*crop \*\*\_Format 1**\*: Boolean **_Format 2_\*\*: width,height

Automatically detect peoples faces and make faces the center of the crop.

If set to **true** (format 1), the optimizer will find all the faces it can detect, make sure that they are in frame and add some padding around them.

If you define _width,height_ as parameters, the optimizer will find all faces it can and focus on the center point between them all (or at the center of one face, if only one is found). Then it will crop an image matching passed _width_ and _height_ with the faces as the center point.
