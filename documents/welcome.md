# Welcome to warrenmd

A clean, Medium-style markdown editor built with **Go** and **Tiptap**.

## Features

- File explorer with folder support
- Rich text editing with markdown output
- Auto-save functionality
- Clean, distraction-free interface

## Getting Started

Click on any file in the sidebar to start editing, or create a new file using the **+** button.

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd + B | Bold |
| Ctrl/Cmd + I | Italic |
| Ctrl/Cmd + S | Save |
